Object Creation Functions, Inheritance, Properties, Methods, Instantiation

Creating:

In JavaScript you can create an object by defining it, a basic example looks like a value pair definition.

var pokemon = {name: "Luxray Lvl X", hitPoints: 110,type: "Electric"}

Using the above object definition, you can see how values can be returned like so:


One nice feature is that you can include a function name in the object itself. In the above example, to display the nicely formatted text about the Luxray, this code was attached to the .innerHTML property:

.innerHTML = pokemon.name +" is an " + pokemon.type + " Pokemon with " + pokemon.hitPoints +"HP"

This could be re-written to include the display function in the code.

Here is the same output as before, just using the function built into the object:

Here is the script to do this, notice how you can use DOT notation to bring up the object parts. Also how this. was used to self reference.

var pokemon1 = {
name: "Luxray Lvl X",
hitPoints: 110,
type: "Electric",
displayPokemon : function() {
return this.name +" is an " + this.type + " Pokemon with " + this.hitPoints +"HP";
}
};
document.getElementById("demo2").innerHTML = pokemon1.displayPokemon() ;

One thing, if when the function is called, if the () are left out, the actual function will print out rather than the desired output. Here is the code missing the ()

document.getElementById("demo2").innerHTML = pokemon1.displayPokemon ;

Inheritance, Instantiation and Constructors

While it is very easy to set up an object, it would not be very fun to create objects all the time using the above methods. It is possible to use a function to create objects and then use the new keyword in the creation.

The object can be created like so:

function CreatePokemon(name, type, hitPoints) {
this.name = name;
this.type = type;
this.hitPoints = hitPoints;
}

// Create a new object named "pika"
var pika = new CreatePokemon("Pikachu", "Electric", 40);

Now to display the name of the new Pokemon, just use to display the name of the new Pokemon.

document.getElementById("demo4").innerHTML = pika.name;

Below is the output:

Since a Pokemon should have a trainer, let's create an object for trainers.

function CreateTrainer(firstName, lastName, beatElite4) {
this.firstName = fistName;
this.lastName = lastName;
this.beatElite4 = beatElite4;
}

// Create a new object named "Ash"
var ash = new CreateTrainer("Ash", "Ketchum", true);

Now let's modify the CreatePokemon object to include a trainer:
function CreatePokemon(name, type, hitPoints, trainer) {
this.name = name;
this.type = type;
this.hitPoints = hitPoints;
this.trainer = trainer;
}

Now with the trainer added to the object, it is possible to access the information from the Trainer object from the Pokemon object.

Using the innerHTML, we can produce this sentence by accessing the elements in the objects.
document.getElementById("demo5").innerHTML = pika.trainer.firstName + " and " + pika.name + " train together!";

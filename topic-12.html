Script to Transform Div boxes

First, create 3 Div boxes, one for each axis; X,Y and Z.
#myDivX {
width: 300px;
height: 100px;
background-color: green;
border: 1px solid black;
}

#myDivY {
width: 300px;
height: 100px;
background-color: red;
border: 1px solid black;
}

#myDivZ {
width: 300px;
height: 100px;
background-color: rgb(179, 255, 0);
border: 1px solid black;
}

Then in the script, the function change is called with an interval of 1 second.
setInterval(change, 1000);

The function gets the current second and multiplies that by 18, just to make it go faster and used that amount as the degree of rotation for each Div.
var seconds = new Date().getSeconds() * 18;

Then each Div has the rotation applied to it.
myDivX.style.transform = "rotateX("+seconds+"deg)";
myDivY.style.transform = "rotateY("+seconds+"deg)";
myDivZ.style.transform = "rotateZ("+seconds+"deg)";

Manipulating CSS Class Properties Using JavaScript

Script to modify CSS

This script is called once per second, when it is called it modifies the class "Lead" in the heading about this paragraph. A couple of things of note, when called the element, instead of using the font-size CSS property, you have to convert it to camel case, fontSize. This applies to pretty much any CSS property in this case. Also, you have to indicate which element you are calling, so you have to use [elementID].

Here is the script used:

setInterval(changeFont, 1000);

function changeFont()
{
var seconds = new Date().getSeconds();
//alert(seconds);
document.getElementsByClassName("lead")[0].style.fontSize= seconds + "px";
}

Loops:

The While Loop

The While Loop uses the following form:
while (condition) {
statement....
}

Here is an example of a While Loop, it will execute until days is equal to less than 1.

Here is the code:
var Days = 5; //Set variable for days left
var vacationText = "";
while (Days > 0) {
vacationText += "Days until vacation: " + Days + "
";
Days--;
} document.getElementById("While-Demo").innerHTML = vacationText;

The Do-While Loop

The Do-While Loop uses the following form:
do {
statement....
} while(condition)

The main difference between the Do-While loop and the While loop is that the Do-While has the conditon evaluator at the end of the loop. As a result the loop will always execute at least once. Using essentially the same code as the While loop with the difference that the vairable Days is already zero, the loop runs once. In a While loop, it would execute at all since the condition is false at the start

Here is an example of a D0-While Loop, days are set to zero to start.

Here is the code:
var newDays = 0; //Set variable for days left
var vacationTextNew = ""; // Empty string to store results
do {
vacationTextNew += "Days until vacation: " + Days + "
";
newDays--;
} while(newDays > 0);
document.getElementById("Do-While-Demo").innerHTML = vacationTextNew;

The For Loop

The For Loop uses the following form:
for (initialization; condition; end-expression ) {
statement...
}

Here is an example of a For Loop, it will execute until days until vacation is zero.

Here is the code:
var vacationTextNewFor = ""; // Empty string to store results
for (var counter = 5; counter >0; counter--) {
vacationTextNewFor += "Days until vacation: " + counter + "
";
}
document.getElementById("For-Demo").innerHTML = vacationTextNewFor;

The If statement

The If statement uses the following form:
if (condition)
{
lines of code to be executed if condition is true
}

Here is an example of an If statement, it will execute if 5>3.

Here is the code:
var ifText = ""; // Empty string to store results
if (5>3) {ifText="The equation 5>3 is true, so you are seeing this message.";}
document.getElementById("If-Demo").innerHTML = ifText;

The Else statement

The Else statement uses the following form:
if (condition)
{
lines of code to be executed if condition is true
}
else
{
lines of code to be executed if condition is true
}

Here is an example of an Else statement, it will execute if 2>3.

Here is the code:
var elseText = ""; // Empty string to store results
if (2>3) {elseText="The equation 2>3 is true, so you are seeing this message.";}
else
{elseText="The equation 2>3 is false, so you are seeing this message.";}
document.getElementById("Else-Demo").innerHTML = elseText;

}
else if

Functions:

A function is a way group a set of statements together, give it a name and allow it to be called to execute the statements as often as needed.

Example:

Here is the code:
function easyOne() {
return "This message was called by a funtion.";
}
document.getElementById("functionDemo").innerHTML = easyOne();

You can invoke the function by calling it from within a script tags, easyOne().

Variables:

Javascript is a loosely typed language, it does not require you to declare what type a variable is. Javascript will reconize the types internally. The folowing are the different types in javascript."

Number - it can hold a 64-bit double precison number. One thing to watch for, var number="22" is actually a string. Var number=22 is a number.

String - holds all string values like, var sentence="This is a string.";

Boolean - can only hold values of true or false.

Undefined - while the variable was created, there is nothing assigned to it, so it is undefined, "var piglet".

Null - intentionally has no value.

Object - an object can also be a variable.

Parameters:

Parameter Rules

  • JavaScript function definitions do not specify data types for parameters.
  • JavaScript functions do not perform type checking on the passed arguments.
  • JavaScript functions do not check the number of arguments received.
  • If a function is called with missing arguments (less than declared), the missing values are set to: undefined

Arrays:

An array can store multiple values in a single variable.
An array can be delared several ways:
var myArray = new Array();
or
var myArray = [];
Arrays can hold any type and can be mixed types. var exampleArray = [44, "cars", null, undefined, "fun"];
Array indexes are zero based, myArray[0] has the value of 44, myArray[4] has a value of fun.

Here is the exampleArray printed out.

Here is the code:
var exampleArray = [44, "cars", null, undefined, "fun"];
document.getElementById("ArrayDemo").innerHTML = exampleArray;

Notice how the elements null and undefined are there, just not displayed.

Associative Arrays:

Associative Arrays are Key Value pairs, it is an abstract data type.

Here is the Associative Array example printed out, in this case the value of CS2.

Here is the code:
var blizzardGames = new Array();
blizzardGames["SC2"] = "StarCraft 2";
blizzardGames["HOTS"] = "Heroes of the Storm";
blizzardGames["WoW"] = "World of Warcraft";
document.getElementById("AssociativeDemo").innerHTML = blizzardGames.SC2;

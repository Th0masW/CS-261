Creating CSS3 Transitions and Animations in CSS and triggering them with JavaScript.

CSS transitions using Javascript:

The original property is set with CSS, using Javascript it is possible to change the properties on the fly.
Below is a rectangle that is black. The container will be called "magic"


Clicking the box will change it's color.

Here is the script that does it. The script modifies the color property with this statement: magic.style.backgroundColor = color
The function changeColor randomly generates a new color which is assigned with each click.

var container = document.querySelector("#container");
var zeros = "0000000";

container.addEventListener("click", changeColor, false);

function changeColor(e) {
var color = "#" + Math.floor(Math.random() * 0xFFFFFF).toString(16);
var colorLength = color.length;

if (colorLength < 7) {
color += zeros.substring(0, zeros.length - colorLength);
}
container.style.backgroundColor = color;
}

CSS animations using Javascript:

CSS Animations can be done using Javascript...
The thing is that CSS is powerful enough that it is questionable if you need to use Javascript very much with it.
It is handy to start/stop the animations, but other than that...


Here is a script that annimates a box dropping down the screen.

Here is the CSS:

#boxy {
width: 400px;
height: 400px;
position: relative;
background: blue;
}
#animate {
width: 50px;
height: 50px;
position: absolute;
background-color: green;
}

And the script:

function myMove() {
var elem = document.getElementById("animate");
var pos = 0;
var id = setInterval(frame, 5);
function frame() {
if (pos == 350) {
clearInterval(id);
} else {
pos++;
elem.style.top = pos + "px";
elem.style.left = pos + "px";
}
}
}

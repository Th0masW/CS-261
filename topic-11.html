Designing, Defining, and Triggering CSS3 Transitions without Custom Libraries

Basic transition of a box

Here is a basic DIV box, click the buttons to change the color.



Here is the CSS for the box container:
div#box1 {
background: blue;
width: 400px;
height: 200px;
}

The JavaScipt uses the "onclick" to change the color by using the transition property:
elem.style.transition = "background 1.0s linear 0s";
 The background is changed over a period of a second in a linear way, the 0s means that the transition starts with zero delay.

HTML5 Tags - Video, Audio, and Canvas

Embed Video in a page




With HTML5, it is possible to embed a video with no add-ons like Flash. The video can be controlled with some simple JavaScript funtions. The above video is forst embeded into a video container, here is the code:
video id="video1" width="500"
source src="videos/pirates.mp4" type="video/mp4"

Then the buttons are added to call and control the playback function:
button onclick="playPause()">Play/Pause /button
button onclick="makeBig()">Big /button
button onclick="makeSmall()">Small /button
button onclick="makeNormal()">Normal /button

The actual script just uses the HTML5 functions to play the video.
var myVideo = document.getElementById("video1");

function playPause() {
if (myVideo.paused)
myVideo.play();
else
myVideo.pause();
}

function makeBig() {
myVideo.width = 600;
}

function makeSmall() {
myVideo.width = 320;
}

function makeNormal() {
myVideo.width = 500;
}

Using Canvas

Canvas is quite powerful, in the past, things like this required Flash or something along those lines to produce dynamic graphics. Color me impressed with what can be done using canvas.

Here is a fairly simple bar graph style graphic. Which will be handy for my project.

Here is the code for it:
function draw() {
var ctx = document.getElementById('canvas').getContext('2d');

// Draw background
ctx.fillStyle = 'rgb(255, 221, 0)';
ctx.fillRect(0, 0, 150, 37.5);
ctx.fillStyle = 'rgb(102, 204, 0)';
ctx.fillRect(0, 37.5, 150, 37.5);
ctx.fillStyle = 'rgb(0, 153, 255)';
ctx.fillRect(0, 75, 150, 37.5);
ctx.fillStyle = 'rgb(255, 51, 0)';
ctx.fillRect(0, 112.5, 150, 37.5);

// Draw semi transparent rectangles
for (var i = 0; i < 10; i++) {
ctx.fillStyle = 'rgba(255, 255, 255, ' + (i + 1) / 10 + ')';
for (var j = 0; j < 4; j++) {
ctx.fillRect(5 + i * 14, 5 + j * 3
7.5, 14, 27.5);
}
}
}draw();

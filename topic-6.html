DOM Manipulation Using createElement, appendChild, insertBefore, removeChild, etc.

Script to createElement, appendChild, insertBefore and removeChild:

click the buttons to createElement and appendChild and the other button to removeChild.







Here is the script used, removeChild is NOT a good name for the function as it seems to be a reserved phrase, hence why the function name is removeChld.

function createAdd()
{
var paragraph = document.createElement("P");
var text=document.createTextNode("This paragraph was created using createElement, then appened to the DIV using appendchild.");
paragraph.appendChild(text);
document.getElementById("createAdd").appendChild(paragraph);
}


function removeChld()
{
var child = document.getElementById("createAdd");
child.parentNode.removeChild(child);
//item.parentNode.removeChild(item);
//document.getElementById("createAdd").removeChild(text);
}
